Сложные Методы Массивов

Содержание

  • map
  • for Each
  • filter
  • reduce
  • findIndex
  • IndexOf
  • find

[ map ]

Метод map() используется для трансформации массива. Он создаёт новый массив, который будет состоять из результатов вызова для каждого элемента

[ for Each ]

Метод for Each() перебирает все элементы массива по порядку, но не создает новый массив.

[ filter ]

С помощью метода filter() можно создать новый массив на основе имеющегося, но мы можем убрать не нужные элементы в массиве с помощью заданных условий

[ reduce ]

Метод массива reduce() используется для вычисления внутри массива какого-нибудь единого значения, которое мы хотим узнать

[ findIndex ]

Метод findIndex() возвращает индекс элемента,которого мы ищем .Если элемента нет, то возвращается -1.

[ indexOf ]

Найти элемент в массиве можно по индексу и еще с какого индекса начать с помощью метода indexOf(). Если по указанному индексу нет элемента, то возвращается -1.

[ find ]

Метод find() ищет один (первый попавшийся) элемент, на котором функция-колбэк вернёт true.Если ничего не найдено возвращается пустой массив
